Redux - Flux arxitecturasina yazilmiw JS kitabxanasidir.
Flux - data idare etmek ucun arxitecturadir 
Flux arxitecturasinda 4 ed element var : store, dispatch, action, view.
Redux Install : npm i @reduxjs/toolkit react-redux
Redux projecte bawladiqda store qovlugu yaradiriq icerisinde index.js yaradiriq icerisine :

import { configureStore } from '@reduxjs/toolkit'
export default configureStore({
  reducer: {}
})
bu kodu elave edirik. Belelikle Store yarandi

App.jsx de esas kodu <Provider store={store}> teqinin arasina aliriq

Slice qovlugu yaradiriq icerisinde CounterSlice.js yaradiriq icerisinde ise createSlice yaziriq ve ozu create edir.
const CounterSlice = createSlice({
    name: "counter"
    ve burda bir bawlangic state vermeliyik
    initialState,
})

yuxarisinda  const initialState = {
    value: 0
} yaradiriq 

daha sonra initialState, altindan 
          reducers: {
        increment: (state) => {
            state.value += 1
        },
        decrement: (state) => {
            state.value -= 1
        }
    }

1 vahid artirma 1 vahid azaltma funksiyasini increment ve decrement ile edirik.
decrement ardinca reset yaradiriq aynen decrement kimi, state.value ise = edirik 0 a 
Daha sonra incrementByAmount yaradiriq, bizden state ve action deyerleri alir ve action i payload etmek lazimdir. : 
            incrementByAmount: (state, action) => {
            state.value += action.payload
        }
sonra CounterSlice.reducer i export edirik
export const { decrement, increment, reset, incrementByAmount } = counterSlice.action; - bunuda export edirik

Slice Reduce i Store a kocurmek
store qovlugunun icerisindeki index.js de reducer in icerisine counterReducer yaziriq ve ozumuz import edirik. 

import { configureStore } from '@reduxjs/toolkit'
import counterReducer from "../slices/counterSlice"

export default configureStore({
  reducer: {
    counter: counterReducer
  }
})  bele bir kod alinir.
